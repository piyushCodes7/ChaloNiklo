A fun and quirky travel agency landing page designed to inspire spontaneous travel! The tagline says it all: **"Socho Mat, Niklo Fatafat!"** (Don't think, just leave quickly!).
You can visit the frontend here **"https://piyushcodes7.github.io/ChaloNiklo/"**

This project was created with a design-first approach, starting from a concept in **Figma** and then brought to life with HTML, CSS, and JavaScript, with inspiration from **Evolve AI**.

## üöÄ Features

* **Vibrant & Animated UI:** A lively, animated gradient background and floating emojis create an engaging and dynamic user experience.
* **Quirky & Fun Content:** The website is filled with humorous text and a playful tone to entertain and connect with users.
* **Interactive Elements:** Fun, interactive UI elements like buttons that trigger funny alerts, and hover effects on cards and navigation links.
* **Responsive Design:** A mobile-friendly layout to ensure a great experience on all devices.
* **Engaging Sections:**
    * **Hero Section:** A catchy tagline and an amusing description to grab the user's attention immediately.
    * **Fun Stats:** A humorous take on the company's "track record."
    * **Trending Destinations:** Beautifully designed cards showcasing popular travel spots with calls-to-action.
    * **Footer:** Includes social media links and a final touch of humor.
